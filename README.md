## 📌 Project Overview
Welcome to the **Global Starbucks Stores & Consumer Behavior** Exploratory Data Analysis (EDA) project. This repository contains a comprehensive data analysis of Starbucks locations worldwide, simulating a real-world corporate business analyst scenario. 

The primary objective of this project is to uncover geographical patterns, understand store ownership distribution, and identify structural anomalies using statistical methodologies and aesthetic data visualizations.

## 🗄️ Dataset
* **Source:** Kaggle (Starbucks Store Locations Worldwide)
* **Description:** The dataset includes information on over 25,000 Starbucks stores operating across the globe, featuring data points like Ownership Type, City, Country, Latitude, and Longitude.

## 🎯 Key Objectives & Methodology
In this project, I performed end-to-end Exploratory Data Analysis covering the following phases:

1. **Data Profiling & Summary Statistics:** * Extracted high-level statistical summaries.
   * Analyzed store density across the top 10 global markets.
2. **Univariate Analysis (Distributions):** * Plotted histograms/bar charts to visualize the distribution of different ownership models (Company Owned vs. Licensed, etc.).
3. **Outlier Detection (Boxplots):** * Mapped the geographical spread (Longitude/Latitude) across ownership types to detect isolated stores and spatial anomalies.
4. **Multivariate Analysis & Correlation:** * Generated a Correlation Heatmap to check dependencies between numerical geographical parameters.
   * Plotted an advanced Pairplot grid to visually recognize overlapping market expansion patterns.

## 💡 Key Business Insights Discovered
* **Ownership Dominance:** Visualized the exact ratio by which Company-Owned and Licensed stores dominate the global market compared to Joint Ventures.
* **Spatial Outliers:** Identified specific stores situated in extreme geographic coordinates, highlighting potential strategic outposts or data anomalies.
* **Clustering Trends:** Observed high-density clusters in specific longitudinal zones, indicating concentrated market saturation in key global regions.
